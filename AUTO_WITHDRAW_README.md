# 🚀 АВТОВЫВОД СРЕДСТВ - ИНСТРУКЦИЯ

## ⚠️ ВАЖНО!

**Автовывод - это мощная функция, которая автоматически переводит найденные средства на ваш адрес!**

- Используйте **ТОЛЬКО на свой страх и риск**
- Проверьте адреса **ДВАЖДЫ** перед включением
- Храните приватные ключи в **БЕЗОПАСНОСТИ**
- Рекомендуется **тестировать** на малых суммах

---

## 📖 КАК ИСПОЛЬЗОВАТЬ

### 1. Включение автовывода через код:

```python
from checkers.crypto_checker import CryptoChecker

checker = CryptoChecker()

# Настройка адресов для вывода
checker.enable_auto_withdraw(
    addresses={
        "ethereum": "0xВАШ_ETH_АДРЕС",
        "bsc": "0xВАШ_BSC_АДРЕС",
        "bitcoin": "bc1ВАШ_BTC_АДРЕС",
        "tron": "TВАШ_TRX_АДРЕС",
        "solana": "ВАШ_SOL_АДРЕС",
    },
    min_amounts={
        "ethereum": 0.01,  # Минимум 0.01 ETH для вывода
        "bsc": 0.01,       # Минимум 0.01 BNB
        "bitcoin": 0.001,  # Минимум 0.001 BTC
        "tron": 10,        # Минимум 10 TRX
        "solana": 0.1,     # Минимум 0.1 SOL
    }
)

# Теперь при проверке приватных ключей или сид-фраз
# средства будут автоматически выводиться!
```

### 2. Проверка с автовыводом:

```python
import asyncio

async def main():
    checker = CryptoChecker()
    
    # Включаем автовывод
    checker.enable_auto_withdraw(
        addresses={"ethereum": "0xВАШ_АДРЕС"}
    )
    
    # Проверяем приватный ключ
    result = await checker.check("0xПРИВАТНЫЙ_КЛЮЧ")
    
    # Если найден баланс - он автоматически выведется!
    if result.get("info", {}).get("auto_withdraw"):
        print("✓ Средства выведены!")
        for tx in result["info"]["auto_withdraw"]:
            print(f"  TX: {tx['tx_hash']}")
            print(f"  Сумма: {tx['amount']}")

asyncio.run(main())
```

### 3. Просмотр лога выводов:

```python
# Получить все выводы
log = checker.get_withdraw_log()
for entry in log:
    print(f"Время: {entry['timestamp']}")
    print(f"Сеть: {entry['chain']}")
    print(f"Сумма: {entry['amount']}")
    print(f"TX: {entry['tx_hash']}")
    print("---")

# Экспорт лога в файл
checker.export_withdraw_log("my_withdrawals.json")
```

### 4. Выключение автовывода:

```python
checker.disable_auto_withdraw()
```

## 🔧 НАСТРОЙКИ

### Минимальные суммы для вывода:

По умолчанию:
- **ETH**: 0.01 ETH (~$30)
- **BSC**: 0.01 BNB (~$6)
- **BTC**: 0.001 BTC (~$40)
- **TRX**: 10 TRX (~$2)
- **SOL**: 0.1 SOL (~$15)

Можно изменить при включении автовывода.

### Оставлять газ:

По умолчанию система **оставляет немного монет на газ** (0.0001 ETH).

Это нужно чтобы транзакция прошла успешно!

## ⚠️ БЕЗОПАСНОСТЬ

### ✅ ЧТО ДЕЛАТЬ:
- Проверяйте адреса **ДВАЖДЫ**
- Тестируйте на **малых суммах**
- Храните лог выводов
- Используйте **свои адреса**

### ❌ ЧЕГО НЕ ДЕЛАТЬ:
- НЕ используйте чужие адреса
- НЕ включайте автовывод без проверки
- НЕ храните приватные ключи в коде
- НЕ делитесь логами выводов

---

## 🐛 УСТРАНЕНИЕ ПРОБЛЕМ

### "Недостаточно средств для газа"
- Баланс слишком мал для покрытия комиссии
- Увеличьте минимальную сумму вывода

### "Адрес не указан"
- Забыли указать адрес для этой сети
- Проверьте `enable_auto_withdraw(addresses={...})`

### "Транзакция не прошла"
- Проверьте баланс газа
- Проверьте правильность адреса
- Проверьте подключение к сети
