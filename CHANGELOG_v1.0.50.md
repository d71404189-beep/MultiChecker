# 📋 Changelog v1.0.50

## 🆕 Новые функции

### ⌨️ Поддержка Ctrl+V для вставки из буфера обмена
- **Вставка в поле ввода данных** - теперь можно нажать Ctrl+V для вставки скопированного текста
- **Вставка в поля автовывода** - все адреса (ETH, BTC, TRX, SOL) поддерживают Ctrl+V
- **Кроссплатформенность** - работает на Windows (Ctrl+V), Mac (Cmd+V), Linux (Ctrl+V)
- **Контекстное меню** - правая кнопка мыши на Entry полях показывает меню "Вставить"
- **Умная вставка**:
  - В Textbox вставляет в позицию курсора
  - В Entry заменяет весь текст
  - Автоматическая обработка ошибок

### 📁 Drag-and-Drop файлов
- **Перетаскивание файлов** прямо в поле ввода данных
- **Автоматическое чтение** содержимого файла
- **Поддержка кодировок**:
  - UTF-8 (по умолчанию)
  - CP1251 (Windows Cyrillic)
  - Автоматическое определение
- **Визуальная обратная связь** - сообщения об успехе/ошибке
- **Опциональная зависимость** - работает без tkinterdnd2 (только без drag-and-drop)

## 🎨 Улучшения UX

### Удобство ввода данных
- ✅ Больше не нужно использовать кнопку "↑ Файл"
- ✅ Просто перетащите файл в окно
- ✅ Или скопируйте адрес и нажмите Ctrl+V
- ✅ Работает во всех полях ввода

### Контекстное меню
- Правый клик на Entry полях
- Опция "📋 Вставить (Ctrl+V)"
- Современный дизайн в стиле приложения
- Автоматическое закрытие при потере фокуса

## 🔧 Технические улучшения

### Новые методы
- `_bind_paste_shortcut(widget)` - добавляет поддержку Ctrl+V
- `_setup_drag_and_drop(textbox)` - настраивает drag-and-drop
- Поддержка как CTkTextbox, так и CTkEntry виджетов

### Обработка событий
- Биндинг `<Control-v>` для Windows/Linux
- Биндинг `<Command-v>` для Mac
- Биндинг `<Button-3>` для контекстного меню
- Биндинг `<<Drop>>` для drag-and-drop

### Обработка ошибок
- Graceful fallback если tkinterdnd2 не установлен
- Обработка разных кодировок файлов
- Валидация путей к файлам
- Информативные сообщения об ошибках

## 📦 Зависимости

### Новые зависимости
- `tkinterdnd2==0.3.0` - для drag-and-drop (опционально)

### Установка
```bash
# Полная установка с drag-and-drop
pip install -r requirements.txt

# Или только обновление
pip install tkinterdnd2==0.3.0
```

**Примечание**: Приложение работает и без tkinterdnd2, но drag-and-drop будет недоступен.

## 🎯 Примеры использования

### Вставка адреса для автовывода
1. Скопируйте адрес кошелька (Ctrl+C)
2. Кликните в поле "ETH/BSC/Polygon"
3. Нажмите Ctrl+V
4. Адрес вставлен! ✓

### Загрузка файла через drag-and-drop
1. Откройте папку с файлом данных
2. Перетащите файл в поле "Входные данные"
3. Файл автоматически загружен! ✓

### Использование контекстного меню
1. Скопируйте текст
2. Правый клик на поле ввода
3. Выберите "📋 Вставить (Ctrl+V)"
4. Текст вставлен! ✓

## 💡 Советы

### Горячие клавиши
- **Ctrl+V** - вставить из буфера обмена
- **Ctrl+C** - скопировать (стандартно)
- **Ctrl+A** - выделить всё (стандартно)

### Drag-and-Drop
- Поддерживаются текстовые файлы (.txt, .csv, .json и т.д.)
- Файл должен быть в кодировке UTF-8 или CP1251
- Максимальный размер файла не ограничен (но будьте осторожны с большими файлами)

### Контекстное меню
- Работает только на Entry полях (адреса автовывода)
- Для Textbox используйте Ctrl+V напрямую
- Меню автоматически закрывается при клике вне его

## 🔐 Безопасность

- Вставка из буфера обмена безопасна
- Файлы читаются только локально
- Нет передачи данных на сервер
- Валидация путей к файлам

## 📝 Известные ограничения

1. **Drag-and-drop требует tkinterdnd2** - без этой библиотеки функция недоступна
2. **Только текстовые файлы** - бинарные файлы не поддерживаются
3. **Контекстное меню только для Entry** - для Textbox используйте Ctrl+V

## 🚀 Производительность

- Вставка из буфера обмена мгновенная
- Загрузка файлов зависит от размера
- Нет влияния на скорость проверки
- Минимальное использование памяти
