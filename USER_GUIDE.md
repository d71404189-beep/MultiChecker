## 🚀 Введение

**MultiChecker Pro** - это мощный инструмент для проверки и анализа криптовалютных кошельков, email, социальных сетей, игровых аккаунтов и AI сервисов.

### Основные возможности:

- ✅ **Crypto Checker** - проверка балансов на 15+ блокчейнах
- ✅ **Email Checker** - валидация email адресов
- ✅ **Social Checker** - проверка аккаунтов соцсетей
- ✅ **Game Checker** - проверка игровых аккаунтов
- ✅ **AI Checker** - проверка AI сервисов
- ✅ **NFT Checker** - поиск NFT коллекций
- ✅ **Airdrop Hunter** - поиск доступных airdrop'ов
- ✅ **DeFi Positions** - анализ DeFi позиций
- ✅ **Auto Withdraw** - автоматический вывод средств
- ✅ **Portfolio Tracker** - отслеживание портфеля
- ✅ **Smart Alerts** - умные уведомления

## 🤖 Автоматизация

### Auto Withdraw (Автовывод)

Автоматический вывод найденных средств на ваш адрес.

**⚠️ ВНИМАНИЕ: Используйте осторожно!**

#### Настройка:

1. Включите переключатель **"Включить автовывод"**
2. Укажите адреса для вывода:
   - **ETH/BSC/Polygon**: 0x...
   - **Bitcoin**: bc1... или 1... или 3...
   - **Tron**: T...
   - **Solana**: ...

3. Установите минимальные суммы:
   - **ETH**: 0.01 (по умолчанию)
   - **BTC**: 0.001
   - **TRX**: 10
   - **SOL**: 0.1

#### Как работает:

1. При обнаружении баланса > минимума
2. Автоматически создается транзакция
3. Средства выводятся на указанный адрес
4. Оставляется небольшая сумма на газ

#### Лог выводов:

Все выводы логируются в файл `auto_withdraw_log.txt`:
```
[2024-01-15 10:30:45] ETH: 0.05 -> 0x... | TX: 0xabc...
[2024-01-15 10:31:12] BTC: 0.002 -> bc1... | TX: abc123...
```

### Auto Swap (Автообмен)

Автоматический обмен токенов на ETH/BNB перед выводом.

#### Настройка:

1. Включите **"Включить автообмен"**
2. Выберите целевую монету: ETH / BNB / MATIC
3. Минимум USD: 1.0 (токены дешевле не обмениваются)
4. Slippage: 1.0% (допустимое проскальзывание)
5. DEX: Uniswap / PancakeSwap / 1inch

#### Поддерживаемые DEX:

- **Ethereum**: Uniswap V2/V3, 1inch
- **BSC**: PancakeSwap, 1inch
- **Polygon**: QuickSwap, 1inch

## ⚙️ Настройки

### Основные настройки:

#### Потоки (Threads)
- **Рекомендуется**: 100
- **Минимум**: 1
- **Максимум**: 1000
- Больше потоков = быстрее проверка, но выше нагрузка

#### Таймаут (Timeout)
- **Рекомендуется**: 10-30 секунд
- Для медленных сетей: 30-60 секунд
- Для быстрых: 5-10 секунд

#### Прокси
Форматы:
```
# HTTP/HTTPS
http://ip:port
http://user:pass@ip:port

# SOCKS4/SOCKS5
socks4://ip:port
socks5://user:pass@ip:port

# Файл с прокси
proxy.txt (один прокси на строку)
```

**Автоматическая ротация** при использовании файла.

### Telegram уведомления:

1. **Token**: токен бота от @BotFather
2. **Chat ID**: ID чата для отправки
3. **Enabled**: включить/выключить

## ❓ FAQ

### Q: Как проверить сид-фразу?

**A:** Просто вставьте 12/15/18/21/24 слова через пробел:
```
word1 word2 word3 word4 word5 word6 word7 word8 word9 word10 word11 word12
```

MultiChecker автоматически:
- Проверит валидность (BIP-39 checksum)
- Сгенерирует первые 10 адресов для каждой сети
- Проверит балансы на всех адресах
- Покажет общую сумму

---

### Q: Поддерживаются ли токены?

**A:** Да! Автоматически проверяются:

**Ethereum:**
- USDT, USDC, DAI, WETH
- LINK, UNI, SHIB, PEPE
- И другие популярные ERC-20

**BSC:**
- USDT, USDC, BUSD
- CAKE, BNB
- И другие BEP-20

**Polygon:**
- USDT, USDC, DAI
- MATIC, WETH
- И другие токены

---

### Q: Как работает автовывод?

**A:** При обнаружении баланса:
1. Проверяется минимальная сумма
2. Рассчитывается газ для транзакции
3. Создается и подписывается транзакция
4. Отправляется в сеть
5. Логируется результат

**⚠️ Важно:**
- Оставляется небольшая сумма на газ
- Работает только с приватными ключами
- Не работает с адресами без ключей

---

### Q: Безопасно ли использовать?

**A:** MultiChecker:
- ✅ Работает локально на вашем ПК
- ✅ Не отправляет данные на сторонние серверы
- ✅ Открытый исходный код
- ✅ Не хранит приватные ключи

**Но:**
- ⚠️ Используйте прокси для анонимности
- ⚠️ Не делитесь найденными ключами
- ⚠️ Автовывод используйте осторожно

---

### Q: Почему не находит баланс?

**Возможные причины:**

1. **Неверный формат** - проверьте правильность адреса/ключа
2. **Сеть недоступна** - попробуйте позже
3. **Прокси блокирует** - отключите прокси
4. **Таймаут мал** - увеличьте до 30 секунд
5. **API лимит** - подождите несколько минут

---

### Q: Как ускорить проверку?

**Советы:**

1. **Увеличьте потоки** до 200-500
2. **Используйте быстрые прокси** или без прокси
3. **Уменьшите таймаут** до 5-10 секунд
4. **Включите кэширование** (автоматически)
5. **Проверяйте пакетами** по 100-1000 адресов

## 🔧 Troubleshooting

### Проблема: "Connection timeout"

**Решение:**
1. Увеличьте таймаут до 30-60 секунд
2. Проверьте интернет соединение
3. Попробуйте другой прокси
4. Отключите прокси

---

### Проблема: "Invalid private key"

**Решение:**
1. Проверьте формат ключа (HEX должен быть 64 символа)
2. Убедитесь что ключ начинается с `0x` (для HEX)
3. Для WIF проверьте что ключ начинается с `5`, `K` или `L`

---

### Проблема: "Rate limit exceeded"

**Решение:**
1. Подождите 5-10 минут
2. Используйте прокси для ротации IP
3. Уменьшите количество потоков
4. Используйте платные API ключи (Etherscan, etc.)

---

### Проблема: "No balance found"

**Это нормально!** Большинство случайных адресов пустые.

**Для тестирования:**
Используйте известные адреса с балансом:
```
# Vitalik Buterin
0xd8dA6BF26964aF9D7eEd9e03E53415D37aA96045

# Binance Hot Wallet
0x28C6c06298d514Db089934071355E5743bf21d60
```

---

### Проблема: Приложение зависает

**Решение:**
1. Уменьшите количество потоков
2. Закройте другие программы
3. Увеличьте RAM (рекомендуется 8 GB)
4. Проверяйте меньшими пакетами

---

### Проблема: "Module not found"

**Решение:**
```bash
pip install -r requirements.txt --upgrade
```

Если не помогло:
```bash
pip install customtkinter aiohttp bip-utils web3 solana tronpy
```
