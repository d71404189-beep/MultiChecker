# 🚀 Release Notes v1.0.50

## Основные нововведения

### ⌨️ Ctrl+V - Вставка из буфера обмена
Наконец-то! Теперь вы можете использовать привычное сочетание клавиш Ctrl+V для вставки текста!

**Где работает:**
- ✅ Поле "Входные данные" - вставляйте списки аккаунтов
- ✅ Поле "ETH/BSC/Polygon" - вставляйте адрес для автовывода
- ✅ Поле "Bitcoin" - вставляйте BTC адрес
- ✅ Поле "Tron" - вставляйте TRX адрес
- ✅ Поле "Solana" - вставляйте SOL адрес
- ✅ Все остальные поля ввода

**Как использовать:**
```
1. Скопируйте адрес кошелька (Ctrl+C)
2. Кликните в нужное поле
3. Нажмите Ctrl+V
4. Готово! ✓
```

**Дополнительно:**
- 🖱️ Правый клик мыши → "📋 Вставить (Ctrl+V)"
- 🍎 На Mac работает Cmd+V
- 🐧 На Linux работает Ctrl+V

### 📁 Drag-and-Drop файлов
Перетаскивайте файлы прямо в окно приложения!

**Как использовать:**
```
1. Откройте папку с файлом
2. Перетащите файл в поле "Входные данные"
3. Содержимое автоматически загружено! ✓
```

**Поддерживаемые форматы:**
- 📄 .txt - текстовые файлы
- 📊 .csv - таблицы
- 📋 .json - JSON данные
- 📝 Любые текстовые файлы

**Кодировки:**
- UTF-8 (по умолчанию)
- CP1251 (Windows Cyrillic)
- Автоматическое определение

## 🐛 Известные проблемы

1. **Linux**: На некоторых окружениях рабочего стола drag-and-drop может не работать
   - **Решение**: Используйте Ctrl+V или кнопку "↑ Файл"

2. **Mac**: Cmd+V может конфликтовать с системными горячими клавишами
   - **Решение**: Используйте контекстное меню (правый клик)

3. **Большие файлы**: Файлы >500MB могут вызвать зависание
   - **Решение**: Разбейте файл на части
