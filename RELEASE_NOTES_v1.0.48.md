# 🚀 MultiChecker v1.0.48 - Auto-Withdraw GUI

## 🎉 ГЛАВНАЯ ФИЧА: Автовывод прямо в интерфейсе!

**Теперь не нужно писать код!** Настройка автовывода доступна прямо в GUI!

---

## ✨ ЧТО НОВОГО:

### 💰 Секция "Автовывод средств" в Crypto чекере

Добавлена новая секция с полным функционалом настройки автовывода:

#### 1. **Переключатель автовывода**
- Включить/выключить одним кликом
- Предупреждение о безопасности

#### 2. **Поля для адресов:**
- **ETH/BSC/Polygon** - один адрес для всех EVM сетей (Ethereum, BSC, Polygon, Avalanche, Base, Arbitrum, Optimism)
- **Bitcoin** - для BTC
- **Tron** - для TRX
- **Solana** - для SOL

#### 3. **Настройка минимальных сумм:**
- ETH: 0.01 (по умолчанию)
- BTC: 0.001
- TRX: 10
- SOL: 0.1
- Можно изменить на любые значения!

#### 4. **Автоматическое применение:**
- При нажатии "Старт" настройки применяются автоматически
- Если найдены средства - они сразу выводятся
- Результат показывается в логе

## 🎯 КАК ИСПОЛЬЗОВАТЬ:

### Шаг 1: Откройте вкладку **Crypto**

### Шаг 2: Найдите секцию **"💰 Автовывод средств"**

### Шаг 3: Включите переключатель

### Шаг 4: Укажите адреса

```
ETH/BSC/Polygon: 0xВАШ_АДРЕС
Bitcoin: bc1... или 1... или 3...
Tron: T...
Solana: ...
```

### Шаг 5: Настройте минимумы (опционально)

### Шаг 6: Вставьте приватные ключи или сид-фразы

### Шаг 7: Нажмите **"▶ Старт"**

**Готово!** Средства будут автоматически выводиться на указанные адреса!

## 🔧 ТЕХНИЧЕСКИЕ ДЕТАЛИ:

### Изменения в коде:

**main.py:**
- Добавлена секция автовывода в `_build_tab()` (только для Crypto)
- Добавлены 9 новых полей в интерфейс
- Обновлена логика в `_run()` для применения настроек
- Динамическое размещение элементов (row зависит от tab_name)

**Новые поля:**
- `auto_withdraw_enabled` - переключатель
- `withdraw_eth`, `withdraw_btc`, `withdraw_trx`, `withdraw_sol` - адреса
- `min_eth`, `min_btc`, `min_trx`, `min_sol` - минимумы

**Логика:**
1. При включенном автовыводе и нажатии "Старт"
2. Считываются все адреса и минимумы
3. Вызывается `checker.enable_auto_withdraw(addresses, min_amounts)`
4. При проверке средства автоматически выводятся
5. Результат показывается в логе
