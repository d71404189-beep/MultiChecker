## 🖼️ 1. NFT CHECKER

### Что делает:
Проверяет NFT коллекции на кошельках и показывает их реальную стоимость.

### Возможности:
- ✅ Поддержка **Ethereum, Polygon, Solana**
- ✅ Интеграция с **OpenSea, Blur, Magic Eden**
- ✅ Автоматический расчет **floor price**
- ✅ Показ **общей стоимости** портфеля NFT
- ✅ Выделение **ценных NFT** (>$100)
- ✅ Группировка по **коллекциям**

### Популярные коллекции:
- **Ethereum:** BAYC, MAYC, Azuki, CloneX, Doodles, Pudgy Penguins, Moonbirds, Otherdeeds, CryptoPunks, Meebits
- **Polygon:** y00ts, DeGods
- **Solana:** Okay Bears, DeGods, y00ts, Mad Lads

### Пример вывода:
```
🖼️ NFT НАЙДЕНО: 15 шт
💰 Общая стоимость: ~$12,450.00

💎 ЦЕННЫЕ NFT (>5 шт):
  • Bored Ape #1234 - $45,000.00
  • Azuki #567 - $8,500.00
  • Pudgy Penguin #890 - $3,200.00

📚 КОЛЛЕКЦИИ:
  • Bored Ape Yacht Club: 2 NFT (~$90,000.00)
  • Azuki: 3 NFT (~$25,500.00)
```

### Как использовать:
1. Проверь адреса в Crypto чекере
2. Нажми кнопку **"🖼️ NFT"**
3. Получи список всех NFT с ценами

## 🪂 2. AIRDROP HUNTER

### Что делает:
Проверяет eligibility (право на получение) аирдропов от популярных проектов.

### Поддерживаемые проекты:
- ✅ **LayerZero** - кросс-чейн протокол
- ✅ **zkSync** - L2 решение
- ✅ **Starknet** - L2 решение
- ✅ **Arbitrum** - L2 сеть
- ✅ **Optimism** - L2 сеть
- ✅ **Base** - L2 от Coinbase

### Критерии проверки:
**LayerZero:**
- Минимум 10 транзакций
- Минимум $100 объем
- Минимум 2 сети

**zkSync:**
- Минимум 5 транзакций
- Минимум $50 объем
- Минимум 3 контракта

**Starknet:**
- Минимум 3 транзакции
- Минимум $10 объем

### Пример вывода:
```
🪂 ДОСТУПНЫЕ АИРДРОПЫ: 3 шт
💰 Примерная стоимость: ~$1,200

🟢 LayerZero: ~$500 (high confidence)
   └─ 25 транзакций через 5 сетей
🟡 zkSync: ~$400 (medium confidence)
   └─ 12 транзакций, 8 контрактов
🟠 Base: ~$300 (low confidence)
   └─ Активность в Base

⏳ ПОТЕНЦИАЛЬНЫЕ АИРДРОПЫ: 2 шт
  • Starknet (60%)
    └─ Нужно еще 2 транзакции
  • Optimism (40%)
    └─ Увеличить активность в Optimism
```

### Как использовать:
1. Проверь адреса в Crypto чекере
2. Нажми кнопку **"🪂 AIRDROP"**
3. Узнай на какие аирдропы eligible

## 📊 3. DEFI POSITIONS CHECKER

- **Lido** (stETH, wstETH) - ~3.5% APY
- **Rocket Pool** (rETH) - ~3.2% APY

**🏦 Lending:**
- **Aave V3** - lending/borrowing
- **Compound V3** - lending/borrowing

**💧 Liquidity Pools:**
- **Uniswap V2/V3** - LP позиции
- **Curve** - стабильные пулы

**🌾 Yield Farming:**
- **Convex** - boosted rewards
- **Yearn** - автоматизированные стратегии

### Пример вывода:
```
💰 ОБЩАЯ СТОИМОСТЬ DEFI: $15,750.00

🔒 СТЕЙКИНГ (2 позиции):
  • Lido stETH: 5.2500 (~$13,125.00) APY: 3.5%
  • Rocket Pool rETH: 1.0000 (~$2,625.00) APY: 3.2%

🏦 LENDING (1 позиция):
  • Aave V3: Supplied $5,000.00 | Borrowed $2,000.00
    └─ Health Factor: 2.50

💧 LIQUIDITY POOLS (1 позиция):
  • Uniswap V3 ETH/USDC: $3,500.00 (Fee: 0.3%)

🌾 YIELD FARMING (1 позиция):
  • Convex 3pool: $2,000.00 (APY: 8.5%)
```

### Как использовать:
1. Проверь адреса в Crypto чекере
2. Нажми кнопку **"📊 DEFI"**
3. Увидь все застейканные средства

## ⚠️ ВАЖНЫЕ ЗАМЕЧАНИЯ

### Rate Limits:
- **OpenSea:** 2 запроса/сек без API ключа
- **CoinGecko:** 10-50 запросов/мин
- **Etherscan:** 5 запросов/сек без API ключа

### Рекомендации:
1. Проверяй не более 10 адресов за раз
2. Используй API ключи для больших лимитов
3. Добавь задержки между запросами
4. Кэшируй результаты для повторных проверок

### Ограничения:
- NFT Checker: только публичные коллекции
- Airdrop Hunter: примерные оценки (не гарантия)
