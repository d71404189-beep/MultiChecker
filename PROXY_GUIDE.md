## 📋 ПОДДЕРЖИВАЕМЫЕ ТИПЫ ПРОКСИ

MultiChecker поддерживает **ВСЕ** популярные типы прокси:

### 1. **HTTP Прокси**
```
http://ip:port
http://user:pass@ip:port
```

### 2. **HTTPS Прокси**
```
https://ip:port
https://user:pass@ip:port
```

### 3. **SOCKS4 Прокси** ✨ NEW!
```
socks4://ip:port
socks4://user:pass@ip:port
```

### 4. **SOCKS5 Прокси** ✨ NEW!
```
socks5://ip:port
socks5://user:pass@ip:port
```

### 5. **Без протокола** (по умолчанию HTTP)
```
ip:port
user:pass@ip:port
```

## 🎯 ФОРМАТЫ

### С авторизацией:

## 💡 КАК ИСПОЛЬЗОВАТЬ

### Способ 1: Один прокси для всех

1. Вставь прокси в поле **"Прокси"**
2. Формат: `socks5://ip:port`
3. Запусти проверку
4. Все запросы пойдут через этот прокси

**Пример:**
```
socks5://123.45.67.89:1080
```

### Способ 2: Список прокси из файла

1. Создай файл `proxy.txt`
2. Добавь прокси (по одному на строку):
   ```
   socks5://123.45.67.89:1080
   socks4://98.76.54.32:1080
   http://11.22.33.44:8080
   ```
3. В поле **"Прокси"** укажи: `proxy.txt`
4. Чекер будет ротировать прокси

### Способ 3: С авторизацией

Если прокси требует логин/пароль:

```
socks5://myuser:mypass@123.45.67.89:1080
http://admin:12345@98.76.54.32:8080
```

## 🔧 УСТАНОВКА ЗАВИСИМОСТЕЙ

Для работы **SOCKS4/SOCKS5** прокси нужна библиотека `aiohttp-socks`.

### Автоматическая установка:
```bash
pip install -r requirements.txt
```

### Ручная установка:
```bash
pip install aiohttp-socks
```

### Проверка установки:
```python
python -c "import aiohttp_socks; print('✅ SOCKS поддержка установлена!')"
```

- ✅ Самые универсальные

- ✅ Поддержка UDP
- ✅ Лучшая анонимность

```bash
# HTTP прокси
curl -x http://ip:port https://api.ipify.org

# SOCKS5 прокси
curl -x socks5://ip:port https://api.ipify.org
```

### Способ 3: Через Python
```python
import requests

proxies = {
    'http': 'socks5://ip:port',
    'https': 'socks5://ip:port'
}

response = requests.get('https://api.ipify.org', proxies=proxies)
print(f"Ваш IP через прокси: {response.text}")
```

## ❌ ЧАСТЫЕ ОШИБКИ

### 1. "aiohttp-socks не установлен"

**Проблема:** Пытаешься использовать SOCKS прокси без библиотеки

**Решение:**
```bash
pip install aiohttp-socks
```

### 2. "Connection timeout"

**Проблема:** Прокси не отвечает или неправильный адрес

**Решение:**
- Проверь IP и порт
- Проверь что прокси работает
- Увеличь таймаут в настройках

### 3. "Proxy authentication required"

**Проблема:** Прокси требует авторизацию

**Решение:**
```
socks5://username:password@ip:port
```

### 4. "Invalid proxy format"

**Проблема:** Неправильный формат прокси

**Решение:**
```
✅ Правильно: socks5://123.45.67.89:1080
❌ Неправильно: 123.45.67.89:1080:socks5
```

## 🔒 БЕЗОПАСНОСТЬ

### Рекомендации:

1. **Используй проверенные прокси**
   - Не используй бесплатные публичные прокси
   - Покупай у надежных провайдеров

2. **Проверяй прокси перед использованием**
   - Убедись что прокси работает
   - Проверь скорость и стабильность

3. **Ротируй прокси**
   - Используй разные прокси для разных аккаунтов
   - Меняй прокси периодически

4. **Не передавай чувствительные данные**
   - Через ненадежные прокси
   - Без шифрования (HTTPS)
