# 📋 Changelog v1.0.49

## 🆕 Новые функции

### 🔄 Автоматический обмен токенов
- **Автообмен ERC-20 токенов** на ETH/BNB/MATIC перед выводом
- **Поддержка DEX**: Uniswap V2/V3, PancakeSwap, QuickSwap, 1inch
- **Настраиваемые параметры**:
  - Целевой токен (ETH, BNB, MATIC)
  - Минимальная сумма в USD для обмена
  - Slippage (проскальзывание цены)
  - Выбор DEX для обмена
- **Автоматическое определение сети** и выбор правильного роутера
- **Approve токенов** перед обменом (если требуется)
- **Расчет минимального выхода** с учетом slippage
- **Логирование всех операций** обмена

### 📊 Расширенная статистика сессии
- **Детальная статистика проверки**:
  - Общее количество проверенных адресов
  - Количество валидных адресов
  - Количество адресов с балансом
  - Общая сумма найденных средств в USD
  - Количество выполненных выводов
  - Количество выполненных обменов
- **Лучшая находка сессии**:
  - Адрес с максимальным балансом
  - Сумма в USD
  - Сеть/блокчейн
- **Статистика по сетям**:
  - Количество адресов по каждой сети
  - Общая сумма по каждой сети
  - Сортировка по убыванию суммы
- **Метрики производительности**:
  - Длительность сессии (часы, минуты, секунды)
  - Скорость проверки (адресов в секунду)
  - Процент успешных находок
- **Экспорт статистики** в JSON файл
- **Красивое отображение** в отдельном окне с графиками

## 🔧 Технические улучшения

### Методы CryptoChecker
- `enable_auto_swap()` - включить автообмен с параметрами
- `disable_auto_swap()` - выключить автообмен
- `_auto_swap_tokens()` - выполнить обмен токенов
- `start_session()` - начать новую сессию статистики
- `update_session_stats()` - обновить статистику по результату
- `end_session()` - завершить сессию
- `get_session_stats()` - получить статистику сессии
- `get_session_summary()` - получить текстовую сводку
- `export_session_stats()` - экспортировать в JSON
- `_format_duration()` - форматировать длительность

### Интеграция в main.py
- Применение настроек автообмена при старте проверки
- Автоматический запуск сессии статистики
- Обновление статистики для каждого результата
- Отображение расширенной статистики для Crypto
- Метод `_show_crypto_extended_stats()` для GUI
- Метод `_export_crypto_stats()` для экспорта

## 📦 Зависимости
- Все существующие зависимости сохранены
- Используется `web3>=7.0.0` для работы с DEX
- Используется `eth_account` для подписи транзакций

## 🐛 Исправления
- Корректное позиционирование элементов GUI (row 4-8)
- Правильная обработка ошибок при обмене токенов
- Валидация параметров автообмена
- Безопасная работа со статистикой при отсутствии данных

## 📝 Примечания
- Автообмен работает только для EVM-совместимых сетей
- Требуется достаточно газа для выполнения обмена
- Slippage защищает от неблагоприятных изменений цены
- Статистика сохраняется только для текущей сессии
- Экспорт статистики создает файл с timestamp

## 🔐 Безопасность
- Все транзакции подписываются локально
- Приватные ключи не передаются на сервер
- Проверка баланса перед обменом
- Расчет минимального выхода для защиты от потерь
- Логирование всех операций для аудита
