## 🔥 ОСНОВНЫЕ ВОЗМОЖНОСТИ:

### 1. **Автовывод средств** 💰
- Автоматический вывод найденных средств на ваш адрес
- Поддержка всех EVM сетей (ETH, BSC, Polygon, Avalanche, Base, Arbitrum, Optimism)
- Настраиваемые минимальные суммы для вывода
- Автоматический расчет газа
- Логирование всех транзакций
- Экспорт лога в JSON

### 2. **Расширенная проверка сид-фраз** 🌱
- Проверка 12-24 слов
- Валидация BIP-39 (английский и русский)
- Расширенная деривация (первые 10 адресов для каждого пути)
- Поддержка 8 монет: BTC, ETH, LTC, DOGE, DASH, BNB, SOL, TRX
- Все форматы BTC (Legacy, SegWit, Native SegWit, Taproot)
- **~100+ адресов проверяется с каждой сид-фразы!**

### 3. **Проверка приватных ключей** 🔑
- HEX формат (с/без 0x)
- WIF формат (compressed/uncompressed)
- Автоопределение формата
- Конвертация между форматами
- Проверка безопасности ключа
- Экспорт для кошельков (MetaMask, Electrum)

### 4. **Мультичейн сканирование** 🌐
- 7 EVM сетей одновременно
- Ethereum, BSC, Polygon, Avalanche, Base, Arbitrum, Optimism
- Параллельная проверка для скорости

### 5. **Расширенная аналитика** 📊
- ERC-20/BEP-20/TRC-20 токены
- NFT (OpenSea, Bitcoin Ordinals)
- Staking (Lido stETH, Rocket Pool rETH)
- DeFi позиции (Uniswap V3)
- DEX аппрувы (Uniswap, 1inch)
- Airdrop eligibility
- Цены в USD с изменением за 24ч
- Gas price tracker
- Wallet age (возраст кошелька)
- Activity score

### 6. **Поддержка 12 криптовалют** 💎
- Bitcoin (все форматы)
- Ethereum + все EVM сети
- Solana (+ SPL токены)
- Tron (+ TRC-20 токены)
- TON
- Cardano
- Litecoin
- Dash
- Monero
- Ripple (XRP)
- Dogecoin
- BNB Chain

### 7. **Проверка бирж** 🏦
- 8 бирж: Binance, Bybit, OKX, Huobi, KuCoin, Gate, MEXC, Bitget
- Формат: url:login:pass или url|login|pass
- API ключи
