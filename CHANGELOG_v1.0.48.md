# 🚀 CHANGELOG v1.0.48

## ✨ ЧТО НОВОГО:

### 💰 Автовывод прямо в GUI!

Теперь **не нужно писать код** чтобы настроить автовывод!

#### Добавлено в интерфейс Crypto чекера:

1. **Переключатель "Включить автовывод"**
   - Просто включи/выключи одним кликом
   - Предупреждение о безопасности

2. **Поля для адресов:**
   - **ETH/BSC/Polygon** - один адрес для всех EVM сетей
   - **Bitcoin** - для BTC
   - **Tron** - для TRX
   - **Solana** - для SOL

3. **Настройка минимальных сумм:**
   - ETH: 0.01 (по умолчанию)
   - BTC: 0.001
   - TRX: 10
   - SOL: 0.1
   - Можно изменить на свои значения!

4. **Автоматическое применение:**
   - При нажатии "Старт" настройки применяются автоматически
   - Если найдены средства - они сразу выводятся
   - Всё логируется

## 🎯 КАК ИСПОЛЬЗОВАТЬ:

### Шаг 1: Откройте вкладку Crypto

**Готово!** Средства будут автоматически выводиться на указанные адреса!

## 🔧 ТЕХНИЧЕСКИЕ ДЕТАЛИ:

### Изменения в коде:

1. **main.py:**
   - Добавлена секция автовывода в `_build_tab()` (только для Crypto)
   - Добавлены поля: `auto_withdraw_enabled`, `withdraw_eth`, `withdraw_btc`, `withdraw_trx`, `withdraw_sol`
   - Добавлены поля минимумов: `min_eth`, `min_btc`, `min_trx`, `min_sol`
   - Обновлены row для всех элементов ниже (динамически для Crypto)
   - Добавлена логика применения настроек в `_run()`

2. **Логика работы:**
   - При включенном автовыводе и нажатии "Старт"
   - Считываются все адреса и минимумы
   - Вызывается `checker.enable_auto_withdraw(addresses, min_amounts)`
   - При проверке средства автоматически выводятся
   - Результат показывается в логе
