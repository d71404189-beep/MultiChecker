# 📋 Changelog v1.0.52

## 🆕 Новые функции

### 🎨 Проверка ВСЕХ ERC-20 токенов
- **Автоматическое определение** всех токенов на кошельке через Etherscan API
- **Не только популярные** - теперь находим ВСЕ токены, даже редкие
- **Фильтр по стоимости** - показываем только токены стоимостью от $0.01
- **Автоматические цены** - получаем цены через CoinGecko API
- **Расчет общей стоимости** - показываем сумму всех токенов в USD
- **Поддержка сетей**: Ethereum, BSC, Polygon

**Как работает:**
1. Сначала проверяем популярные токены (USDT, USDC, WETH и т.д.) - быстро
2. Затем получаем список ВСЕХ токенов через Etherscan API
3. Проверяем баланс каждого токена
4. Получаем цены и фильтруем по минимальной стоимости
5. Показываем результат с общей суммой

### 🖼️ Проверка NFT с оценкой стоимости
- **Детальная информация** о каждом NFT через OpenSea API
- **Floor price** - показываем минимальную цену коллекции
- **Оценка в USD** - конвертируем ETH в доллары
- **Топ коллекция** - показываем самую дорогую NFT
- **Общая стоимость** - сумма всех NFT портфеля
- **Поддержка сетей**: Ethereum, Polygon, BSC

**Пример вывода:**
```
5 NFT (~$12,450.00) | Top: Bored Ape Yacht Club
```

### 📊 Экспорт в Excel с форматированием
- **Красивое форматирование** - цветовое кодирование, шрифты, границы
- **Автоширина колонок** - все данные видны полностью
- **Заголовки с цветом** - синий фон, белый текст
- **Статус с цветом**:
  - ✓ Valid - зеленый фон
  - ✗ Invalid - красный фон
- **Структурированные данные**:
  - Номер, Тип, Адрес, Статус, Баланс USD, Детали, Время

**Как использовать:**
1. Завершите проверку
2. Нажмите кнопку "EXCEL" в секции экспорта
3. Файл сохранится как `valid_YYYYMMDD_HHMMSS.xlsx`

## 💡 Советы

### Для максимальной эффективности
1. **Получите API ключи** (бесплатно):
   - Etherscan: https://etherscan.io/apis
   - OpenSea: https://docs.opensea.io/reference/api-keys
   
2. **Установите переменные окружения**:
   ```bash
   set ETHERSCAN_API_KEY=your_key_here
   set OPENSEA_API_KEY=your_key_here
   ```

3. **Используйте Excel экспорт** для анализа:
   - Сортировка по балансу
   - Фильтрация по типу
   - Графики и сводные таблицы

### Ограничения API
- **Etherscan**: 5 запросов/сек (бесплатный план)
- **OpenSea**: 2 запроса/сек
- **CoinGecko**: 10-50 запросов/мин

Если превысите лимиты, получите ошибку 429. Подождите минуту и попробуйте снова.

## 📊 Производительность

### Время проверки
- **Без новых функций**: ~2 сек на адрес
- **С проверкой всех токенов**: +2-3 сек
- **С проверкой NFT**: +1-2 сек
- **Итого**: ~5-7 сек на адрес

### Оптимизация
- Кэширование уменьшает время на 30-50%
- Параллельные запросы ускоряют проверку
- Фильтр по стоимости убирает мусорные токены

## 🔐 Безопасность

- Все API запросы через HTTPS
- API ключи хранятся в переменных окружения
- Нет передачи приватных ключей на сервер
- Кэш хранится только в памяти (не на диске)

## 📝 Известные ограничения

1. **Проверка всех токенов** работает только для EVM сетей (Ethereum, BSC, Polygon)
2. **NFT проверка** требует OpenSea API (может быть недоступен в некоторых регионах)
3. **Excel экспорт** требует установки openpyxl
4. **Цены токенов** могут быть неточными для очень редких токенов
