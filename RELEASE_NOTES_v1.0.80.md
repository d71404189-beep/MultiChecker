## 📦 Установка

### Обновление с v1.0.79:

```bash
# 1. Обновите код
git pull origin main

# 2. Установите новые зависимости (опционально)
pip install plyer win10toast

# 3. Запустите приложение
python main.py
```

### Новая установка:

```bash
# 1. Клонируйте репозиторий
git clone https://github.com/d71404189-beep/MultiChecker.git

# 2. Установите зависимости
cd MultiChecker
pip install -r requirements.txt

# 3. Запустите
python main.py
```
