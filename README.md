# Multi Checker Pro

Desktop-приложение для проверки аккаунтов, кошельков и данных.

## Возможности

- **Email** - Проверка почты (Gmail, Yahoo, Outlook, Proton, iCloud, Mail.ru, Yandex, Rambler, Tutanota)
- **Social** - Проверка соцсетей (Telegram, Discord, Instagram, Twitter, Facebook, TikTok, Reddit)
- **Crypto** - Проверка криптокошельков (Bitcoin, Ethereum, Solana, Tron, Litecoin, Dash, Monero, Ripple) и бирж
- **Games** - Проверка игровых аккаунтов (Steam, Epic Games, Rockstar, Riot, Ubisoft, Blizzard)
- **AI** - Проверка AI сервисов (ChatGPT, Gemini, Claude, Midjourney, Hugging Face, Character AI)

## Установка

```bash
git clone https://github.com/d71404189-beep/MultiChecker.git
cd MultiChecker
pip install -r requirements.txt
python main.py
```

## Использование

1. Выберите вкладку (Email, Social, Crypto, Games, AI)
2. Введите данные для проверки (по одному в строке)
3. Настройте потоки и таймаут
4. Нажмите "Start Check"
