---

### 🔐 Auth Accounts Export

**Модуль:** `checkers/auth_export.py`

**Что это?**
Теперь можно **экспортировать только те аккаунты, в которые можно войти** (есть seed фраза, приватный ключ, email:password или API ключи) + показывается баланс!

**Новая кнопка в интерфейсе:**
```
🔐 AUTH - экспорт аккаунтов с возможностью авторизации
```

---

## 🎯 ВОЗМОЖНОСТИ

### 1. Автоматическое определение аккаунтов с авторизацией

Система автоматически находит аккаунты где есть:

✅ **Seed фраза** (12-24 слова)
```
word1 word2 word3 ... word12
```

✅ **Приватный ключ** (HEX, WIF)
```
0x1234567890abcdef...
KwDiBf89QgGbjEhKnhXJuH7...
```

✅ **Email:Password**
```
user@example.com:password123
```

✅ **API ключи** (для бирж)
```
API Key + API Secret
```

---

### 2. Отображение баланса

Для каждого аккаунта показывается:
- 💰 **Баланс в USD**
- 🔐 **Тип авторизации**
- 📊 **Данные аккаунта** (адрес, сеть, и т.д.)
- 🔑 **Credentials для входа**

---

### 3. Множественный экспорт

При нажатии кнопки **🔐 AUTH** создается **5 файлов**:

1. **`auth_accounts_detailed_YYYYMMDD_HHMMSS.txt`**
   - Детальный отчет со всеми данными
   - Все аккаунты (с балансом и без)

2. **`auth_accounts_balance_YYYYMMDD_HHMMSS.txt`**
   - Компактный формат
   - Только аккаунты с балансом > $0.01

3. **`auth_credentials_only_YYYYMMDD_HHMMSS.txt`**
   - Только credentials (для быстрого импорта)
   - Формат: одна строка = один credential

4. **`auth_accounts_YYYYMMDD_HHMMSS.json`**
   - JSON формат со всеми данными
   - Для программной обработки

5. **`auth_accounts_YYYYMMDD_HHMMSS.csv`**
   - CSV формат для Excel
   - Колонки: Address, Balance, Auth Type, Credentials

## 🎯 ПРИМЕРЫ ИСПОЛЬЗОВАНИЯ

### Пример 1: Базовое использование

1. Запустите проверку аккаунтов
2. Дождитесь завершения
3. Нажмите кнопку **🔐 AUTH**
4. Получите 5 файлов с результатами

### Пример 2: Программное использование

```python
from checkers.auth_export import AuthAccountExporter, ResultsAnalyzer

# Анализируем результаты
analyzer = ResultsAnalyzer()
exporter = analyzer.analyze_results(results)

# Статистика
stats = exporter.get_statistics()
print(f"Найдено аккаунтов: {stats['total_accounts']}")
print(f"С балансом: {stats['with_balance']}")
print(f"Общий баланс: ${stats['total_balance_usd']:,.2f}")

# Экспорт только с балансом > $100
exporter.export_to_txt(
    "high_balance.txt",
    min_balance=100.0,
    format_type="detailed"
)

# Экспорт только seed фраз
seed_accounts = exporter.filter_by_auth_type("seed")
print(f"Seed аккаунтов: {len(seed_accounts)}")
```

### Пример 3: Фильтрация по балансу

```python
# Только аккаунты с балансом > $1000
high_balance = exporter.filter_by_balance(min_balance=1000.0)

# Аккаунты с балансом от $100 до $1000
medium_balance = exporter.filter_by_balance(
    min_balance=100.0,
    max_balance=1000.0
)

# Экспорт
exporter.export_to_json("high_balance.json", min_balance=1000.0)
```

## 🔍 АВТОМАТИЧЕСКОЕ ОПРЕДЕЛЕНИЕ

Система автоматически определяет тип авторизации:

### 1. Seed фраза
- Проверяет наличие полей: `seed`, `mnemonic`, `phrase`
- Валидирует длину (12-24 слова)
- Извлекает derivation path если есть

### 2. Приватный ключ
- Проверяет поля: `privkey`, `private_key`, `key`
- Валидирует длину (≥ 64 символа)
- Поддерживает HEX и WIF форматы

### 3. Email:Password
- Проверяет наличие обоих полей: `email` и `password`
- Валидирует формат email

### 4. API ключи
- Проверяет поля: `api_key` и `api_secret`
- Используется для бирж (Binance, Bybit, etc.)

---

## 💰 РАСЧЕТ БАЛАНСА

Баланс определяется автоматически:

1. **Прямое значение:** `balance_usd` в результатах
2. **Конвертация:** Если есть `balance` + `currency`:
   - ETH → USD (× $2,500)
   - BTC → USD (× $50,000)
   - Другие → как есть
3. **По умолчанию:** $0.00 если баланс не найден
