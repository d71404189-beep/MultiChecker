## 📋 DUMP PARSER

**Модуль:** `checkers/dump_parser.py`

### ЧТО ЭТО?

**Умный парсер дампов** - автоматически распознает и извлекает данные из дампов любого формата!

### 🎯 ПОДДЕРЖИВАЕМЫЕ ФОРМАТЫ

#### 1. **Email:Password:Seed**
```
user@mail.com:pass123:word1 word2 word3 ... word12
admin@test.com:qwerty:word1 word2 word3 ... word12
```

#### 2. **Email|Password|Privkey**
```
user@mail.com|pass123|0x1234567890abcdef...
admin@test.com|qwerty|0xabcdef1234567890...
```

#### 3. **Email;Password;Address**
```
user@mail.com;pass123;0x742d35Cc6634C0532925a3b844Bc9e7595f0bEb
admin@test.com;qwerty;bc1qxy2kgdygjrsqtzq2n0yrf2493p83kkfjhx0wlh
```

#### 4. **Смешанный формат**
```
email@test.com:password123
word1 word2 word3 word4 word5 word6 word7 word8 word9 word10 word11 word12
0x1234567890abcdef1234567890abcdef1234567890abcdef1234567890abcdef
user@mail.com:pass:0xabcdef...
```

#### 5. **С табуляцией**
```
user@mail.com	pass123	word1 word2 ... word12
admin@test.com	qwerty	0x1234567890abcdef...
```

### 🔍 ЧТО УМЕЕТ НАХОДИТЬ

Парсер автоматически распознает:

1. **📧 Email адреса**
   - `user@mail.com`
   - `admin@test.com`
   - Любые валидные email

2. **🔒 Пароли**
   - Любые строки между разделителями
   - Автоматическое определение

3. **🌱 Seed фразы**
   - 12, 15, 18, 21, 24 слова
   - BIP39 формат
   - Автоматическая валидация

4. **🔑 Приватные ключи**
   - HEX формат (64 символа)
   - С префиксом `0x` или без
   - WIF формат (Bitcoin)

5. **📍 Адреса кошельков**
   - Ethereum: `0x...`
   - Bitcoin: `bc1...`, `1...`, `3...`
   - Автоматическое определение

### 🎯 РАЗДЕЛИТЕЛИ

Поддерживаются все популярные разделители:
- `:` (двоеточие)
- `|` (вертикальная черта)
- `;` (точка с запятой)
- `	` (табуляция)
- `,` (запятая)

**Автоматическое определение разделителя!**

### 🎯 КАК ИСПОЛЬЗОВАТЬ

#### Способ 1: Через кнопку

1. Вставь дамп в поле ввода
2. Нажми кнопку **"📋 ДАМП"**
3. Парсер автоматически:
   - Определит формат
   - Извлечет данные
   - Покажет статистику
   - Подготовит для проверки

#### Способ 2: Программно

```python
from checkers.dump_parser import DumpParser

parser = DumpParser()

# Парсим дамп
dump = """
user1@mail.com:pass123:word1 word2 ... word12
user2@mail.com:pass456:0x1234567890abcdef...
"""

parsed_data = parser.parse_dump(dump)

# Извлекаем для чекера
for_checker = parser.extract_for_checker(parsed_data)

# Статистика
stats = parser.get_stats()
print(f"Найдено seed: {stats['found_seeds']}")
print(f"Найдено privkey: {stats['found_privkeys']}")
```

## 🔧 ТЕХНИЧЕСКИЕ ДЕТАЛИ

### Dump Parser:

**Алгоритм работы:**

1. **Определение разделителя**
   - Подсчет частоты каждого разделителя
   - Выбор самого частого
   - Fallback на пробелы

2. **Извлечение данных**
   - Regex для email
   - Regex для seed (12-24 слова)
   - Regex для privkey (64 hex)
   - Regex для адресов

3. **Валидация**
   - Проверка длины seed
   - Проверка формата privkey
   - Проверка формата адресов

4. **Приоритизация**
   - Seed > Privkey > Address
   - Лучшее для проверки
