# 🚀 Release Notes v1.0.49

## Основные нововведения

### 🔄 Автоматический обмен токенов
Теперь MultiChecker может автоматически обменивать найденные ERC-20 токены на основные монеты (ETH, BNB, MATIC) перед выводом средств!

**Возможности:**
- ✅ Поддержка популярных DEX (Uniswap, PancakeSwap, QuickSwap)
- ✅ Настраиваемый slippage для защиты от потерь
- ✅ Минимальная сумма для обмена (избегаем мелких транзакций)
- ✅ Автоматический выбор оптимального роутера
- ✅ Полное логирование всех операций

**Как использовать:**
1. Откройте вкладку **Crypto**
2. Найдите секцию **"🔄 Автообмен токенов"**
3. Включите переключатель
4. Выберите целевой токен (ETH/BNB/MATIC)
5. Укажите минимальную сумму в USD
6. Настройте slippage (рекомендуется 1-3%)
7. Выберите DEX
8. Запустите проверку

**Пример:**
```
Настройки:
- Цель: ETH
- Минимум: $10
- Slippage: 1%
- DEX: Uniswap

Результат:
✓ Обменял 1500.000000 USDT на 0.428571 ETH
✓ Обменял 250.000000 LINK на 0.071429 ETH
```

## Безопасность

### Меры предосторожности
- 🔒 Все транзакции подписываются локально
- 🔒 Приватные ключи не покидают ваш компьютер
- 🔒 Проверка баланса перед каждой операцией
- 🔒 Защита от проскальзывания цены (slippage)
- 🔒 Полное логирование для аудита

### Рекомендации
- Используйте slippage 1-3% для стабильных токенов
- Устанавливайте адекватный минимум USD (избегайте мелких транзакций)
- Проверяйте адреса вывода перед запуском
- Сохраняйте логи всех операций
- Тестируйте на небольших суммах

## Известные ограничения

1. **Автообмен работает только для EVM-сетей** (Ethereum, BSC, Polygon и т.д.)
2. **Требуется достаточно газа** для выполнения swap транзакций
3. **Статистика сохраняется только для текущей сессии** (не персистентна)
4. **DEX должны быть доступны** в выбранной сети
